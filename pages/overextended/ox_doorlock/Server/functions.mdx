# Funções

Obtém os dados de uma porta com o ID fornecido, correspondendo ao ID da entrada no banco de dados.

## getDoor

```lua
exports.ox_doorlock:getDoor(doorId)
```

Obtém os dados de uma porta com o `id` fornecido, correspondendo ao ID da entrada no banco de dados.

- id: `number`

Retorno:

- door: `table`

## getDoorFromName

```lua
exports.ox_doorlock:getDoorFromName(name)
```

Obtém os dados de uma porta com o `name` fornecido, correspondendo ao nome da entrada no banco de dados.

- name: `string`

Retorno:

- door: `table`

## editDoor

```lua
exports.ox_doorlock:editDoor(doorId, data)
```

Edite a configuração para o `doorId` fornecido.

- doorId: `number`
- data: `table`

### setDoorState

```lua
exports.ox_doorlock:setDoorState(doorId, state)
```

Define uma porta com o `doorId` fornecido como trancada se o estado for `true` ou `1`.

- doorId: `number`
- state: `0` or `1` or `boolean`
